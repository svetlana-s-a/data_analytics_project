# Исследование рынка общественного питания Москвы

## Описание проекта

Цель исследования — проанализировать рынок общественного питания Москвы, выявить факторы, влияющие на популярность заведений, и определить особенности различных категорий ресторанов и кафе для принятия решения об открытии заведения инвестором.

---

## Цель проекта

Провести исследовательский анализ данных (EDA), изучить структуру рынка общественного питания Москвы и определить, какие характеристики могут быть связаны с успешностью заведений.

---

## Стек

- Python

---
## Данные

Датасет содержит информацию о заведениях общественного питания Москвы:

- название заведения;
- категория;
- район;
- рейтинг;
- средний чек;
- количество посадочных мест;
- режим работы;


---

## Что было сделано

- выполнена предобработка данных;
- обработаны пропущенные значения;

- приведены типы данных;

---

## Основные результаты

В результате исследования были выявлены особенности различных категорий заведений общественного питания, различия между районами Москвы и факторы, влияющие на рейтинг и ценовую категорию заведений.

Полученные результаты позволяют использовать данные для принятия решений при выборе концепции нового заведения или анализе конкурентной среды.
