# Анализ пользовательского поведения сервиса «Яндекс Афиша»

Исследовательский анализ данных (EDA), статистическая проверка гипотез и разработка интерактивного дашборда для анализа пользовательского поведения сервиса **«Яндекс Афиша»**.

## Цель проекта

Проанализировать изменения в поведении пользователей в период **с 1 июня по 31 октября 2024 года**, исследовать сезонные изменения спроса на мероприятия и проверить статистические различия между пользователями мобильных устройств и десктопных версий сервиса.

## Данные

Использовались три датасета:

- **orders** — информация о заказах билетов;
- **events** — информация о мероприятиях;
- **exchange rates** — курс тенге к рублю за 2024 год.

---

## Предварительный анализ данных

Перед началом исследования была проведена первичная проверка качества данных на уровне базы данных с использованием SQL.

Выполнены:

- поиск пропусков;
- проверка дубликатов;
- анализ корректности данных;
- предварительная оценка структуры таблиц.

Полный SQL-скрипт находится в файле **[Afisha.sql](Afisha.sql)**.

## Выполненная работа

### Предобработка данных

- обработка пропусков;
- удаление дубликатов;
- преобразование типов данных;
- объединение таблиц;
- приведение выручки к единой валюте;
- расчет дополнительных признаков;
- выделение сезонов продаж.

### Исследовательский анализ данных (EDA)

Проведен анализ:

- динамики продаж;
- сезонности;
- категорий мероприятий;
- возрастных ограничений;
- среднего чека;
- активности пользователей;
- региональных различий;
- эффективности билетных операторов.

### Проверка статистических гипотез

Для сравнения пользователей мобильных устройств и десктопной версии выполнены:

- проверка распределений;
- t-тест Уэлча.

Проверены гипотезы о различиях:

- среднего количества заказов;
- среднего времени между заказами.

# Основные результаты

## 📈 Сезонность

- интерес к новогодним мероприятиям начинает резко расти уже в сентябре;
- осенью увеличивается спрос на семейный контент (0+ и 12+).

**Рекомендация:** запускать продвижение зимних мероприятий уже в начале осени.

---

## 💰 Маркетинговые кампании

Выявлены периоды, когда рост количества заказов сопровождался снижением среднего чека.

**Рекомендация:** провести анализ эффективности акционных кампаний.

---

## 🌍 Региональная аналитика

Обнаружены регионы с высоким потенциалом роста:

- Медовская область;
- Североярская область.

Для каждого региона предложены отдельные рекомендации.

---

## 🤝 Партнеры

Один из билетных операторов обеспечивал высокое количество мероприятий, но приносил значительно меньшую долю выручки.

**Рекомендация:** пересмотреть условия сотрудничества.

---

## 📱 Mobile vs Desktop

Статистический анализ показал:

- мобильные пользователи совершают больше заказов;
- различия статистически значимы;
- интервалы между повторными заказами также отличаются.

Полученные результаты могут использоваться для персонализации коммуникаций и оптимизации мобильного приложения.
