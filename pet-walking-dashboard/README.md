# Операционный дашборд сервиса по выгулу питомцев

Интерактивный дашборд для мониторинга ключевых операционных показателей сервиса по выгулу питомцев.

Проект основан на реальном бизнес-кейсе сервиса по выгулу домашних животных. При разработке использовался опыт управления проектом: структура дашборда, набор метрик и сценарии анализа соответствуют ежедневным задачам операционного менеджмента.

Проект носит демонстрационный характер. Все персональные данные, идентификаторы клиентов и исполнителей заменены случайными значениями. Адреса, объемы заказов и тарифная сетка изменены или сгенерированы случайным образом. Структура данных и логика показателей сохранены.

# 📈 Основные показатели

Дашборд включает мониторинг следующих KPI:

- Общая выручка
- Количество клиентов
- Средний чек
- Количество исполнителей
- Количество выполненных услуг

---

# 📊 Аналитические блоки

### Динамика показателей

- выручка и количество выполненных услуг
- количество обращений
- количество незакрытых заказов

Для графиков доступен выбор уровня агрегации:

- day
- week
- month
- quarter

---

### Структура продаж

- распределение заказов по типам:
  - разовые
  - долгосрочные
  - срочные

---

### Каналы обращений

Сводная таблица обращений по источникам:

- мобильное приложение
- Telegram
- WhatsApp
- телефон
- чат на сайте

---

### География заказов

Для анализа пространственного распределения спроса реализованы:

- интерактивная карта выполненных услуг;
- сводная таблица по районам.

Для каждого района отображаются:

- количество выгулов
- количество исполнителей
- количество заказов
- количество незакрытых заказов

Это позволяет выявлять районы с недостаточным количеством исполнителей и принимать решения по подбору персонала.
